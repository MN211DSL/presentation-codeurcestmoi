J'ai utilisé VS Code pour créer ce site Web

Pourquoi devriez-vous également utiliser VS Code pour apprendre la programmation?

  • - La contenance est assez légère
  • - Moins de consommation de RAM
  • - Fonctionnement puissant sur 3 plateformes : Windows, Linux et macOS
  • - Prise en charge de plusieurs langages de programmation
  • - Simple à éditer, construire et déboguer
  • - Assistance de la communauté GitHub
  • - Téléchargement Gratuit
